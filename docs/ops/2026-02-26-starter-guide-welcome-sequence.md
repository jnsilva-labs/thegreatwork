# Starter Guide Welcome Sequence (Draft)

Lead magnet: 7 Hermetic Principles Starter Guide
Primary goal: orient beginners, build trust, transition into weekly letters
Tone: invitational, clear, non-deterministic, practice-focused

## Email 1 — Deliver the guide + orient the reader
Subject ideas:
- Your Hermetic Principles Starter Guide is here
- Begin here: the 7 Hermetic Principles (free guide)

Purpose:
- Deliver what was promised
- Set expectations for the next few emails
- Establish the site as a living library and practice path

Core points:
- Link to guide download/delivery asset
- Brief explanation of how to use the guide (not just read it once)
- Link to `/start-here`
- Mention weekly letters (without heavy push)

CTA:
- Start Here page

## Email 2 — What the Hermetic principles are (without cliches)
Subject ideas:
- What the Hermetic principles actually are
- A clear way to begin with Hermetic thought

Purpose:
- Build clarity and trust
- Correct common misconceptions (vague mysticism / abstract-only reading)

Core points:
- Principles as operating lenses, not belief tests
- Distinguish source / interpretation / practice
- Encourage choosing one principle to practice this week

CTA:
- `/principles`

## Email 3 — How to use Awareness Paradox as a study path
Subject ideas:
- How to use this site (and where to go next)
- Your beginner path: principles, astrology, practice

Purpose:
- Reduce overwhelm
- Route readers into the content ecosystem intentionally

Core points:
- Path 1: principles
- Path 2: astrology for self-understanding
- Path 3: meditation/practice (journey/tools as they expand)
- Explain that depth unfolds in layers

CTAs:
- `/astrology`
- `/journey`
- `/letters`

## Email 4 — A first practice (reflection prompt email)
Subject ideas:
- A 10-minute Hermetic reflection practice
- Try this today: one principle, one prompt

Purpose:
- Move from reading to practice
- Build engagement and replies

Core points:
- One simple practice prompt (e.g., Mentalism + attention)
- Invite journaling or brief reflection
- Ask for a reply with what they noticed (optional but valuable)

CTA:
- Reply to email OR revisit `/principles/[slug]`

## Email 5 — Join the weekly letters + what comes next
Subject ideas:
- What comes next (weekly letters + deeper study)
- If you want to go deeper, here is the next step

Purpose:
- Transition subscriber from lead magnet flow into ongoing relationship
- Set cadence expectations

Core points:
- What the weekly letters include (Hermetic principles, astrology, practice prompts, tools)
- Reassure no spam / easy unsubscribe
- Mention future tools and guides

CTA:
- `/letters`

## Segmentation Notes (Lightweight)
Tag based on clicked links:
- principles -> `beginner-hermetic`
- astrology -> `astrology-self-understanding`
- journey/practice -> `meditation-practice`

## Copy Rules (Brand Integrity)
- No false urgency
- No deterministic claims in astrology/tarot framing
- Clear next steps, not pressure-heavy sales language
- Distinguish historical source, interpretation, and practice when relevant
